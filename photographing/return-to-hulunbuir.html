重返呼伦贝尔

2022-07-30

我第一次去呼伦贝尔是在高中毕业后，那时候金老师领着我们一群十七八岁的孩子从三河直奔草原腹地，想起来是一件很特别的事。彼时我还是个毛头小子，满脸覆盖着一层青春末期的油，每个毛孔都喊着要迎接新的生活。我不在乎自己是不是还没有准备好，我只是享受着追随时间向前走动，而第一站就是强健、广阔、青绿，充斥着蓦然与苦涩，却又未曾谋面的草原。

从那时向后流转，整整九年。我像一匹马，偶尔环顾嘶鸣，更多的是在埋头咀嚼生活的五味杂陈。我四处奔波，我迷于风尘，我筋疲力竭。我想，该有这样一种强烈的声音出现了——回去看看吧。

说是「回去」，好像归乡一般，也没什么道理，因为我并非生长于草原，说是「声音」，其实也并无依据，无非只是一次计划旅行，而 c 又想去个她没去过的地方罢了。这一切只是在顺理成章地发生。也许有很多事情不是当下可以回味的：那之后再过一年我才写下这篇文章，我也意识到，面对着匀速、沉默的时间，我已不能再说出享受一类的词了。

但旅途还是愉悦的。

我们在一个下午从北京飞往满洲里，落地时已经入夜，在机场出口处处理完核酸检测、防疫信息填报等一系列流程后，就算正式抵达。当时才下过小雨，微风，到处都是湿润的草味，满洲里机场门口的几个猛犸塑像也一并渲染着异于北京的凉爽气息。啊！我仿佛见到了一个阔别多年的旧友，想要开口说些什么却又每每顿住，我激动于重逢的惊喜，又羞怯于询问那些似乎久已存在的陌生的改变……

但是天色已经太晚了，我和 c 只好打车前往旅馆，并一路将自己的纷乱心绪扩散至这广阔的天地间。

c，于满洲里机场

满洲里

可能是因为前一晚下过雨，第二天的天气相当好，站在北京少见的「蓝天白云」下，我们都对接下来的旅程充满了期待。

计划中第一个要去的景区是满洲里国门，虽然是国门，但其实也离市区并不远，因为这个小县城本来就地处边境……国门景区有一座观光塔，在塔顶的观景台能远远地望见对面的俄罗斯小镇。现在回想起来，其实观景台没有什么娱乐的空间，但无尽的绿色、巨大的云朵和奇异的建筑还是给久居城市的我们带来了巨大的视觉冲击，此时的我已经把什么都忘了，只顾着按下快门和来回更换镜头。「看！」我对 c 说，「那边真好看！套娃广场！哦，这边也好看！国门下面有铁路！好像有火车来了……」

国门

从塔上下来后还可以近距离参观国门、进到门里面，内里倒是没什么风景，只有些历史宣传、老气的纪念品商店和并不干净的玻璃窗，我们走马观花似地转了一圈，就出了景区，准备回到城市里漫游。

满洲里的交通还算便利，步行、公交、出租车任君选择。由于没什么目的地，我们就选择了「腿着」，满洲里的大路干净而开阔，两侧都是些看起来稍显落寞的建筑。我偏爱这种冷清，便能无视疲惫地去探寻各种「意外之喜」：最远处是云，近一些是树林，更近一些是被格桑花衬托得朴实和可爱的一切。c 虽然和我路线一致，眼里的景色却并不完全和我重叠——互相分享、互相等待，我们在这段不长的路上走了很久。

不知名的小房子

晚饭是俄罗斯菜，饭后甜点是不远处的小小的北湖公园。

北湖公园，顾名思义，是围绕着北边的一个湖建的公园，看起来是市民们散步和歇脚的地方。地方不大，却显得婉约和精致，让人不禁幻想起这里也许曾发生过的美丽故事二三。湖面倒映着比天空更通透的蓝和白，偶尔有鸭子游过，作为色彩与空间的自然点缀。我们陶醉于此，却又不敢过分停留，生怕破坏了别人眼中的这一份静逸呈现。

北湖公园

这一天的终点是中苏金街，金街并非仅是意义上的「金」，而是入夜后真的会变成金色——夜游于此，不免有些身处异国他乡之感。我和 c 自东向西、又从南向北地走了一圈。可能是受防疫政策的影响，街上并不十分热闹，只有些零散的水果和烧烤摊位，所以除了金色的灯光，也没有什么真正吸引我们的地方。这一天走路很多，早已经累了，我们就决定回去休息，第二天前往下一个目的地。

海拉尔

在诺敏塔拉奶茶店（锅茶和肉非常好吃，强力推荐）吃过午饭后，我们收拾行李去火车站坐火车，一路经过扎赉诺尔、呼和诺尔，沿着海拉尔河，我们抵达了海拉尔市区。对我来说这是一段过分熟悉的路程，因为数年前我第一次来到这里时，正是沿着相反的路线行进的。海拉尔是呼伦贝尔的核心市区，也是去呼伦贝尔其他地方的最好的中转站（呼伦贝尔市面积极大，约等于山东和江苏两个省的和），所以某种意义上这里是我们的必经之地，我们之后的行程也确实都是以海拉尔为中心放射的。

但是在海拉尔市区并没有太多的行程计划——我们在此停留了三天，更多的只是在享受逃离了北京的这份闲散。唯二去过的能称得上是景点的地方，大概就是成吉思汗广场和海拉尔国家森林公园。

成吉思汗广场就在市区中心，离我们落脚的酒店也不远，就成了我们每天茶余饭后散步的地方。我隐约记得上次来呼伦贝尔时，第一站也是这里，只不过当时天色已晚，手机又没有很强的夜摄能力，所以就想不起太多细节了。这次再来，把广场前前后后转了好几遍。实话说并无太多景色可言，但各种各样的人可不少：唱歌的、跳舞的、无所事事的。我和 c 也很乐在其中，可能是因为这里是市民们的「人气」聚集地吧。

我们俩第一次来成吉思汗广场，是到达海拉尔那一天的傍晚，当时气温不算太高，又正赶上漂亮的火烧云。之后又在闷热的晚上来过、暴晒的中午来过、在对面的医院做核酸检测而顺路来过、从别的地方打车回旅馆路过过……我想，大概总要有些不同的角度，才能让自己对眼前的事物更好地了解一番。凭借于此，我和 c 似乎在不约而同地假想定居于海拉尔的琐碎生活，我们沿着河边缓缓走动、在路边小摊买便宜的水果、随心地在苍蝇小店吃晚饭……也许，在什么都特别快的现代社会，为自己虚构几天慢节奏的生活也算得上是旅行的意义之一吧。

至于海拉尔国家森林公园之行，则是在日光暴晒与蚊虫侵扰中度过的。我不甚了解各类树木，于是最有趣的项目成了骑双人自行车……森林公园内还有一个小小的动物园，里面有散养的小羊若干，就不过多记述了。

小鸟树屋，海拉尔国家森林公园

额尔古纳

体会过海拉尔的悠然后，我们在一个下着雨的上午坐大巴前往额尔古纳。

我也曾经到访过额尔古纳，但一直到大巴车抵达终点——拉布大林汽车站，这个小县城本身对我而言还是充满了未知。我和 c 撑着伞，按手机的导航路线一路走到旅馆，然后点了份外卖，稍作休息。下午雨停了，于是我们又重新出去走动。

实话说，这里的街道与传统的北方新村并无二致：普通的学校、普通的工地、普通的商店、普通的居民区。与其说是在感受一个如何壮美的湿地边的落脚小镇，倒更像是在自己农村的老家附近散步那般休闲。再走远一些，无非也就是鸡鸣狗叫，栅栏篱笆。可能最明显的的不同在于这边的屋子外面常会包有一层木材吧。

路边的小家，额尔古纳

而这样的印象在第二天到达湿地景区后被迅速打破——终于第二次来到了这个曾给予我很大震撼的地方，我却无论如何也说不出什么感想了。什么感时伤秋、吟诗作对全都忘了，眼中所见只有壮丽的风景与浓郁的颜色，又仿佛时间被压缩一般，强劲的风速反而把远处大地上的起伏线条衬托成一种安静的流淌或动态的粘稠。我朝着两个相反的方向，用长焦镜头拍下两张全景图。

但在旅程结束后我也仔细想过：如果当时我没有拿着照相机、或是我没有把拍照当成主要目的之一，我还会暂时忘掉那些事情吗？我会想些、说些、记录下些什么呢？我想，我大概也在不自觉中改变了许多。如同杭盖乐队的『回到你身旁』那张专辑的主题曲被命名为「拜访」一般 ，一年后的我再看到这两张照片，竟然也能察觉到一些当时目盲于其中的陌生感了。

那天晚上依旧是在县城里漫游，我随着地图找到一个小小的「哈萨尔广场」（哈萨尔是成吉思汗的弟弟），周围的建筑随着夜深逐渐熄灯后，这里能看到不少星星，我赶紧喊 c 来一起用三脚架拍了几张。可惜我们两个都没什么经验，没拍出特别好看的星星的照片来，不过能看到、拍到（和在没人的空地耍宝），就已经让我们很开心了。

哈萨尔广场的新手星星

在额尔古纳的第三天的行程也是从这里开始的。因为我们发现这个小广场身处县城中心，又离做核酸检测的地方很近……而且我们没有车，所以只好碰运气式的打车前往下一个目的地：额尔古纳国家湿地公园。这个湿地公园据说是新建的，不同于老的湿地景区，这里面积很大，又离市区远，所以只能开车往返。我们本来计划着如果实在交通不便就不去这里了，直接坐公共汽车返回海拉尔。但所幸最后还是打到了车，也因此收获了不少意外之喜。

相比于老的湿地景区，额尔古纳国家湿地公园更像是一个传统意义上的景点了：有载客往返的景区内部的小车、有一些步行路线、有山、有水、有奇奇怪怪的游乐项目和一些圈养的动物，虽然看起来尚未完全建成，但好在风景不错，空气也足够洁净和湿润。在爬山的过程中听到有别的旅行团的导游介绍说：「这里是额尔古纳湿地和大兴安岭的交界。」便一时间来了兴致，很想看看从小就在课本上百闻的大兴安岭究竟是什么样。当然了，最后也没看出什么名堂来，只是环视着拍了不少照片。

c，额尔古纳国家湿地公园的湿地一侧

而后我们错过了去海拉尔的最后一班公交，所以决定还是打车前往。我们在路过陈巴尔虎旗的时候看到了难以置信的鲜红的日落。那景象实在难以言表，它甚至让我开始感慨当日一切行程之机缘巧合：我是多么的幸运啊！草原上原来还有这样热情的颜色啊！

陈巴尔虎旗的日落

尾声

——我们又回到了海拉尔，至此行程已经过去接近一周，我的心情也逐渐缓和，逐渐从追忆或寻道似的探求，转变成了旅行者的愉悦与享受，新的体验慢慢填满了我的内心。这几天来，不论是风景、美食，还是一路见闻的人和事，都让我觉得舒适，便颇有种自以为是的迷途知返之感。但对 c 来说，她还未曾见过那种最经典的草原体验，于是我们的最后一站定在了离市区不远的呼和诺尔。

说呼和诺尔是景区，其实也只是被圈起来的一大片近水的草地，里面放养了一些小羊，拴着不少马和骆驼，有许多这样那样的游乐设施。那天风很大，但我们还是玩得很尽兴，把骑马、射箭、躺在草上小睡这些「草原刻板印象」挨个体验了一遍，可能这也不失为是一种纯粹的轻松快乐：在青黄相接的草原上或随意扫视、或与女友四目相对而笑、或远眺着尽头之外的尽头，偶尔看见了好的构图，还能把相机抬起来点上几张，这样不必动脑地度过几个小时大概也是很难得的吧！

呼和诺尔景区的工作人员，和他的骆驼

c 在摆弄她的小相机

在这最后一部分行程结束、返回海拉尔市区的时候，仿佛是非要补全我们的观景体验一样，呼伦贝尔又给了我们一个惊喜——我们路过了海拉尔河：季节、时间和光线都刚刚好的湛蓝的海拉尔河。与骇人的风不同，海拉尔河看起来渐进而缓慢，而我从未见过这样的蓝色：纯粹、具有极强的侵略性、难以推测深浅，让人羞于舞文弄墨。河水平稳，仔细观察之下又显得颇有暗力，仿佛是涟漪在拉动狂风前进一般……

尾声之后

那之后就是吃吃喝喝、随意走走、买些纪念品，再去成吉思汗广场兜上一圈。睡醒一觉，然后打车前往机场。

我其实在回京的飞机上并没有什么思绪，也没有和 c 交谈太多，只是在考虑回去之后要处理的这些那些工作。至于「重返呼伦贝尔」一事，似乎也已经没有太多仍需追问的意义……我在年少时读过一篇文章，讲述了一个男性跨越十年的物是人非的故事，那时我想，这很有趣，这样沉重的两头着墨岂不是在有意地否定此人这十年间的人生。现在我又有了些额外的想法：但愿我能均匀地审视我余生的今明后天，好让我下一次「追忆十年」的想法来得更轻柔一些。
