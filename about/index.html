终于折腾得差不多了……

本博客，或者说个人站，或者说网络文字垃圾桶……

本垃圾桶建桶至今历经多次搬迁，用过好几个博客平台，考虑了一大堆有的没的因素，最终还是决定搬来 GitHub Pages 上（Hugo 真香紫薯布丁）。

2020-07-04

情绪有点失控，胡乱写了点，又把这个博客的备份拿出来了，想了想还是重新上传吧，写了老半天没人看得到也太寂寞了。

另外在翻 19 年底到现在的草稿时，再一次觉得我的备忘录真是个优质的小素材库，以后可以挑一挑汇总一下并进博客里来。

新换了个 Ezhil theme，tag 有 bug 没搞明白，故暂时关闭。

2019-10-15 ☠️

图文混排有个问题：

把图片设置成 float 的时候图片是和后面一个块顶端对齐的，所以代码就必须放在想要并列放置的文字前边。这样电脑或板子看都正常，但用手机看（网页宽度小于等于 520px）的时候图片就跑到并列文字的上边去了，阅读顺序就比较让人挠头……

太菜，改了一会并没解决，决定在日期后面标个小骷髅头。
